# Project Context: AI News Aria

이 프로젝트는 매일 전날 AI 관련 뉴스, 제품 업데이트, 논문/연구, 정책/규제, 커뮤니티 반응을 모아 오리지널 캐릭터 `아리아`가 설명하는 만화형 카드뉴스로 제작하고, 카카오톡 나와의 채팅으로 보내기 위한 작업 공간입니다.

## 핵심 방향

- 추가 OpenAI API 과금 없이 Codex/GPT 자동화 사용량 안에서 진행한다.
- 1차 버전에서는 n8n과 MCP를 쓰지 않는다.
- 카카오톡 전송은 Kakao OAuth + Kakao REST API `나에게 보내기` 방식으로 한다.
- 카드뉴스 이미지는 카카오톡 전송을 위해 공개 URL이 필요하다.
- 공개 이미지 URL은 무료 방식인 GitHub Pages 또는 다른 무료 호스팅으로 연결할 예정이다.
- 사용자는 컴퓨터가 꺼져 있어도 자동 작업이 진행되길 원한다. 현재 로컬 자동화만으로는 이 조건을 만족하기 어려우므로, `OFFLINE_AUTOMATION_PLAN.md`의 클라우드 실행 구조를 함께 검토한다.

## 캐릭터 설정

캐릭터 이름: 아리아

아리아는 AI 뉴스 큐레이터 역할의 오리지널 미소녀 캐릭터다. 스타일은 과한 가챠풍이 아니라 밝고 깔끔한 현대 애니메이션 뉴스 해설자 느낌을 유지한다.

기준 이미지:

```text
assets/character/aria_reference.png
```

캐릭터 유지 규칙:

- 은청색 단발
- 별 모양 헤어핀
- 흰 셔츠와 네이비 재킷
- 작은 태블릿
- 밝은 뉴스룸 배경
- 과도한 장신구, 무기, 노출, 너무 어두운 분위기는 피한다.

## 카드뉴스 형식

권장 카드 규격:

```text
1080 x 1350 PNG
```

기본 구성:

```text
카드 1: 표지
카드 2-6: 주요 AI 뉴스 해설
카드 7: 커뮤니티 반응 또는 개발자 관점
카드 8: 오늘 주목할 포인트
```

각 카드에 들어갈 정보:

- 장면 설명
- 아리아 표정/포즈
- 말풍선 대사
- 보조 캡션
- 핵심 뉴스 제목
- 중요도
- 출처명/출처 링크

## 생성물 저장 규칙

매일 결과물은 날짜별 폴더에 저장한다.

```text
output/cardnews/YYYY-MM-DD/
```

예시:

```text
output/cardnews/2026-05-01/card_01.png
output/cardnews/2026-05-01/card_02.png
output/cardnews/2026-05-01/cards.json
```

임시 작업물은 `work/drafts/`에 둔다.

공개 URL로 발행한 파일이나 발행용 복사본은 `output/published/`에 둔다.

오래된 결과물은 `scripts/cleanup_outputs.ps1`로 정리한다.
