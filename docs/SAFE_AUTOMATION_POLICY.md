# AI News Aria GPT 자동화 금지 기준

이 문서는 뉴스 출처를 제한하기 위한 문서가 아닙니다. 핵심 목적은 GPT Plus, ChatGPT, Codex를 비공식적으로 자동 조작하다가 계정 리스크가 생기는 일을 막는 것입니다.

## 핵심 원칙

- GPT Plus, ChatGPT, Codex 화면을 브라우저 자동화로 몰래 호출하지 않습니다.
- OpenAI API, 이미지 생성 API, 유료 LLM API를 자동 호출하지 않습니다.
- 뉴스, 블로그, 커뮤니티, 검색 결과 등은 별도 웹 수집 로직으로 확인할 수 있습니다.
- 자동 요약은 GPT 없이 규칙 기반 추출 요약으로 처리합니다.
- 카드뉴스 이미지는 직접 만든 레이아웃, 자체 아이콘, 주제별 도식, 또는 사용 가능한 외부 이미지를 조합합니다.
- 카카오 전송은 사용자가 직접 발급한 Kakao OAuth 토큰으로 `나에게 보내기`만 수행합니다.

## GPT 없이 가능한 자동화

```text
매일 08:00 KST
→ 뉴스/블로그/커뮤니티 등 웹 정보 수집
→ 제목/요약/출처 URL 정리
→ 규칙 기반으로 핵심 항목 선정
→ 카드뉴스 PNG 생성
→ GitHub Pages에 반영
→ Kakao 나에게 보내기
```

## 안전 모드에서 하지 않는 것

- ChatGPT Plus 세션 자동 로그인
- ChatGPT 웹 UI 자동 입력/출력 수집
- Codex 사용량을 자동 API처럼 우회 사용
- 유료 API 키를 숨겨서 자동 과금하는 구조
- OpenAI rate limit, 사용량 제한, 보호 장치 우회

## 리스크에 대한 현실적인 답

정지나 약관 리스크를 "전혀 없다"고 보장할 수는 없습니다. 다만 이 기준은 비공식 GPT 자동 호출을 제거하는 것이 목적입니다. 즉, GPT Plus를 자동으로 긁거나 ChatGPT 화면을 봇처럼 조작하지 않습니다.

완전 자동 요약 품질을 GPT 수준으로 높이려면 공식 OpenAI API가 필요하고, 그 경우에는 별도 API 비용이 발생합니다. 추가 비용 없이 자동화하려면 GPT 없는 규칙 기반 요약으로 운영합니다.
